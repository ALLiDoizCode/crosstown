# @crosstown/core

Core library for Nostr-based ILP peer discovery and SPSP.

## Install

```bash
npm install @crosstown/core
```

## Usage

```ts
import {
  NostrPeerDiscovery,
  NostrSpspClient,
  NostrSpspServer,
  SocialTrustManager,
  BootstrapService,
} from '@crosstown/core';
```
